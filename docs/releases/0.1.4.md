# pytest-warmup 0.1.4

This release republishes the cleaned-up debug and producer model under the correct package version.

## Highlights

- Scoped snapshot bundles through `--warmup-snapshot`
- Targeted snapshot fragments through `prepare(snapshot_id=...)` and `--warmup-snapshot-for`
- Session-level snapshot parsing with producer-scope materialization
- Package-scoped producer support
- Better fail-fast behavior for unused snapshot targets and unsafe debug outputs under pytest-xdist

## Verification

- local test suite
- package build checks
- `twine check dist/*`
